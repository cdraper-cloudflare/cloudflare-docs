Access analytics provide Cloudflare One customers with data on how Access is protecting their network.

Go to Access analytics by navigating to [Zero Trust](https://one.dash.cloudflare.com) > **Analytics** > **Access**.

Customers can view the following data and filters in Access analytics:

**Zero Trust data:**

* Applications accessed
* Failed logins
* Connected users

**Logins over time:**

* Total count of all logins per day
* Filter to see logins for a specific application

**Applications and users:**

* Top applications accessed
* Top connected users

**Time filters:**

* Last hour
* Last 24 hours
* Last 7 days
* Last 30 days
* Current calendar month
